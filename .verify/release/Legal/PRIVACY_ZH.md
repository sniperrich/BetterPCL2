# PCL-N 隐私保护政策

**最后更新日期：2026年6月11日**
**生效日期：2026年6月11日**
**版本号：v2.0**

## 1. 引言与定义
欢迎使用 PCL-N。本隐私政策适用于 PCL-N 客户端软件及其配套云端服务（统称为本服务）。
* **我们或开发者**：指 PCL-N 的独立开发者 MuXue1230-owo。
* **个人数据**：指与您关联并可识别您身份的任何信息。
* **设备**：指您运行 PCL-N 客户端的硬件设备（如 PC）。

本政策旨在说明我们如何收集、使用和共享您的个人数据，以及您如何控制这些数据。

## 2. 我们收集的数据
我们收集的数据取决于您与本服务的交互方式。我们仅遵循最小必要原则收集提供核心功能所必需的数据。

### 2.1 您主动提供的数据
* **凭证与授权令牌**：当您使用微软账户登录时，我们通过 OAuth 2.0 流程获取 Access Token 和 Refresh Token。**我们绝不收集或存储您的明文密码。**
* **用户生成内容 (UGC)**：您通过更衣橱功能上传的本地离线皮肤图片文件，以及您在好友聊天功能中输入的文本内容。
* **同步配置**：您选择上传至云端的启动器 UI 设置、内存分配参数等 JSON 格式数据。

### 2.2 自动收集的设备与网络数据
* **网络标识符 (敏感数据)**：当您使用P2P 联机大厅时，我们的信令服务器会自动收集您的**公网 IP 地址 (IPv4/IPv6) 和 UDP/TCP 端口**。此数据仅用于在您和对端玩家之间建立点对点连接。
* **设备遥测与环境信息**：操作系统版本、CPU 架构、Java 运行时环境 (JRE) 路径及版本、可用物理内存。此数据仅在本地用于适配游戏运行环境，或在您主动提交崩溃日志时上传。
* **本地存储与 Cookie**：我们使用 Windows DPAPI (Data Protection API) 在本地加密存储您的 OAuth Token。内嵌的 WebView2 浏览器控件会使用本地 Cookie 维持微软登录会话，这些 Cookie 不会上传至我们的服务器。

## 3. 我们如何使用数据
我们使用收集到的数据来提供和改进本服务：
* **提供核心功能**：使用您的 IP 地址和端口进行 P2P 信令交换；使用您的 Token 向 Mojang/Microsoft 验证游戏所有权。
* **安全与完整性**：使用 mTLS (双向 TLS) 客户端证书指纹来验证请求来源，防止第三方恶意脚本滥用我们的云端 API。
* **通信**：在您的设备与我们的云端服务器之间路由聊天消息和状态同步数据。

## 4. 数据共享与披露
我们**不会**将您的个人数据出售给任何第三方。我们仅在以下特定情况下共享数据：
* **P2P 联机对手**：在您主动加入或创建联机房间时，您的公网 IP 地址和端口将直接共享给特定的对端玩家。
* **官方服务提供商**：您的 Token 和游戏 UUID 会发送至 Microsoft / Mojang 官方 API 以完成正版验证。
* **法律要求**：若收到具有管辖权的法院、公安机关的合法传票或法律命令，我们可能会披露必要的日志或数据。

## 5. 数据安全与存储
* **存储位置**：您的云端同步数据、聊天记录和离线皮肤存储于**中国大陆**境内的服务器中。
* **传输安全**：所有客户端与服务器之间的通信均强制使用 TLS 1.2/1.3 加密，并辅以 mTLS 证书校验。
* **数据保留**：
  * **聊天记录**：作为即时通讯缓存，默认保留 7 天后自动物理删除。
  * **IP 地址**：信令服务器仅在内存中短暂保留您的 IP 以维持会话，断开连接后立即清除，不作持久化日志记录。
  * **非活跃账户**：若您的账户连续 365 天未与云端服务交互，我们保留匿名化或删除您的云端同步数据的权利。

## 6. 您的权利与控制
根据《中华人民共和国个人信息保护法》(PIPL)，您享有以下权利：
* **访问与导出**：您可以在客户端设置中查看当前云端存储的所有配置数据。
* **删除与注销**：您可以随时在客户端执行清除云端数据操作。该操作将不可逆地永久删除您在服务器上的所有同步配置、皮肤和好友关系。

## 7. 儿童隐私
本服务不面向 14 周岁以下的儿童。我们不会故意收集儿童的个人信息。若我们发现在未经监护人同意的情况下收集了儿童的数据，我们将立即删除相关数据。

## 8. 政策的变更
我们可能会不定期更新本政策。重大变更将通过客户端首次启动时的弹窗或 GitHub 仓库的公告通知您。继续使用本服务即表示您接受更新后的政策。
